##概要

rails練習用コピーアプリ

##対象アプリ

「stackoverflow」

##作成機能

・質問一覧表示
　・タブ切り替え（アクティブ、人気、今週、今月）
　・タグ絞り込み
・質問回答詳細
　・回答新規、編集
　・△▽☆✔︎付け、外し
　・コメント追加、削除
・新規質問
・ログイン

##作成対象外機能

・StackExchange
・ツアー、ヘルプ
・質問検索（上メニュー）
　・質問
　・タグ
　・ユーザー
　・バッジ
　・未回答
・プロフィール

##セットアップ

migrate後にseedで初期データ登録
